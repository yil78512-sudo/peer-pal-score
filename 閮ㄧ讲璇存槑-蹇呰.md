# 部署说明

当前文件夹是 Firebase Firestore 版，不再使用 Supabase。

## 本地预览

```bash
npm install
npm run dev
```

学生端：`http://localhost:8080/#/student`

管理端：`http://localhost:8080/#/admin`

如果终端显示端口是 5173，就把 8080 改成 5173。

## Firebase

Firebase 配置已经写入 `src/lib/peerData.ts`。数据会保存到 Firestore：

- `ratings`：评分记录
- `portfolios`：作品集链接

Firestore 规则请参考 `firebase-firestore-rules.txt`。

## GitHub Pages

后续发布到 GitHub Pages 后，学生端和管理端通常是：

- `https://你的用户名.github.io/peer-pal-score/#/student`
- `https://你的用户名.github.io/peer-pal-score/#/admin`
